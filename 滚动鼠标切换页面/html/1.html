SCROLLIFY

Power steering for your scroll wheel

A jQuery plugin that assists scrolling and smoothly snaps to divs.

Fully configurable and optimised for touch.

Basic Setup

Scrollify requires jQuery 1.6+ and an easing library such as jquery.easing.js.

		<! doctype html>
		<html>
		<head>
			<script>
				$(function() {
					$.scrollify({
						div : "div",
					});
				});
			</script>
		</head>
		<body>
			<div></div>
			<div></div>
		</body>
		</html>

Configuration

		$.scrollify({
		div : "div",
		divName : "div-name",
		easing: "easeOutExpo",
		scrollSpeed: 1100,
		offset : 0,
		scrollbars: true,
		before:function() {},
		after:function() {}
		});

Options

section
A selector for the divs.
sectionName
Scrollify lets you define a hash value for each section. This makes it possible to permalink to particular sections. This is set as a data attribute on the sections. The name of the data attribute is defined by 'divName'.
easing
Define the easing method.
offset
A distance in pixels to offset each divs position by.
scrollbars
A boolean to define whether scroll bars are visible or not.
before
A callback that is called before a div is scrolled to via the move method. Arguments include the index of the div and an array of all div elements.
after
A callback that is called after a new div is scrolled to. Arguments include the index of the div and an array of all div elements.

Methods

The move method can be used to scroll to a particular div. This can take the index of the div, or the name of the div preceded by a hash.

		$.scrollify("move","#name");
